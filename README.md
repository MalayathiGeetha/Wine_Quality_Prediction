
# 🍷 Wine Quality Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a wine is of **Good Quality** or **Bad Quality** using Machine Learning algorithms. The Wine Quality dataset is analyzed, preprocessed, and used to train multiple classification models. The models are evaluated and compared to identify the best-performing algorithm.

---

## 🎯 Objective

- Perform data preprocessing and exploratory data analysis (EDA).
- Convert the wine quality score into a binary classification problem.
- Train and evaluate different Machine Learning models.
- Compare model performance.
- Improve the best model using Hyperparameter Tuning.
- Analyze feature importance.

---

## 📂 Dataset

- **Dataset:** Wine Quality Dataset
- **Target Variable:** `quality_label`
  - **Good (1):** Quality ≥ 7
  - **Bad (0):** Quality < 7

## 📊 Project Workflow

1. Load the dataset
2. Perform Exploratory Data Analysis (EDA)
3. Check for missing values
4. Correlation analysis using Heatmap
5. Create binary target variable
6. Split dataset into training and testing sets
7. Train Logistic Regression without scaling
8. Evaluate model performance
9. Apply StandardScaler
10. Train Logistic Regression with scaling
11. Compare Logistic Regression, KNN, and Decision Tree
12. Perform Hyperparameter Tuning using GridSearchCV
13. Analyze Feature Importance
14. Draw conclusions
